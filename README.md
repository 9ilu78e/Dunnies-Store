# Dunnies Store

Dunnies Store is a modern e-commerce application built with Next.js and TypeScript. This project provides a seamless shopping experience for users while offering an intuitive admin dashboard for managing products and orders.

## Features

- **Public Pages**: 
  - Product listing and detail pages
  - Cart and checkout functionality
  - User account management

- **Admin Dashboard**: 
  - Manage products and orders
  - View analytics and reports

- **API Integration**: 
  - RESTful API for products, orders, and authentication

- **Reusable Components**: 
  - Modular UI components for consistent design

- **Custom Hooks**: 
  - Simplified state management and API calls

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd dunnies-store
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   - Copy `.env.example` to `.env` and configure your environment variables.

4. **Run the development server**:
   ```bash
   npm run dev
   ```

5. **Open your browser**:
   - Navigate to `http://localhost:3000` to view the application.
